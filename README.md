# Dhananjay Sode — Developer Portfolio

A modern, responsive portfolio website with a dark/light theme, glassmorphism UI, and animated effects.

## Pages
- `index.html` — Hero / landing
- `about.html` — Profile, skills, and background
- `projects.html` — Project showcase
- `contact.html` — Contact form

## Tech
- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- Material Symbols icons, Inter & JetBrains Mono fonts

## Features
- Dark / light theme toggle
- Scroll fade-in animations, particle background, 3D tilt, custom cursor
- Fully responsive layout

## Structure
```
portfolio_modern/
├── *.html          # Pages
├── css/            # Styles (theme, effects, per-page)
├── js/             # Scripts (theme, effects, per-page)
└── assets/         # Images & favicon
```

## Run locally
Open `index.html` in a browser, or serve the folder:
```bash
python3 -m http.server
```
